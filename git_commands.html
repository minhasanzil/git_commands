MOST IMPORTANT COMMANDS

git config user.name 'username' // config username for yout git version contrl

git config user.email 'email' // config email for your git version control

git init //initialise local git repo

touch 'filename.format' // make a new file

git add // add files to index

git status // check status

git commit // commit changes

git push http://address.git //push to remote repo

git pull htp://address.git // pull latest from remote repo

git clone http://address.git // clone repo into a new directory

git remote set-url origin http://address.git //add link to your remote repo

git remote -v // chek which remote repo are you saving with

git push origin master // push to the remote repo

git log - // eg: git log limit -5 **limits log by 5 commits**

git log -p // prints all the changes made with user

git diff //show unstaged changes

UNDO CHANGES

git revert //Create new commit that undoes all of the changes made in , then apply it to the current branch.

REMOTE REPOSITORIES

git remote add //Create a new connection to a remote repo. After adding a remote, you can use as a shortcut for in other commands.

git fetch // fetches a specific branch from repo
